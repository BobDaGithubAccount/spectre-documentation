Introduction

This Silver DofE project is game engine written in Java, which has a basic rendering pipeline written in LWJGL 2.X, using OpenGL. The goal is for it to support the source engine map format so the hammer editor can be the map editor, to have basic multiplayer functionality and to have server side modding capability. This project was made by ME

Spectre-Client

This is the game-client, that renders graphics and is the actual game instance. Its source code can be found here.

Spectre-Server

This is the game-server, that hosts gamelogic, forwards packets, enables multiplayer and server side modding. Hopefully its source code can be optimized in due course. You can find it here.
